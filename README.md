# Basel-Problem
manim animation of Basel-Problem

这个项目使用 [Manim 社区版](https://www.manim.community/) 制作动画，直观展示欧拉在 1735 年证明巴塞尔问题（即 ζ(2)=π²/6）的经典思路。

## 📽️ 动画内容

动画分步展示了欧拉的核心思想：

1. **泰勒展开**：展示 sin x 的泰勒级数，并提取 x² 项的系数 -1/6
2. **无穷乘积**：基于 sin x 的根，将其表示为无穷乘积形式
3. **系数比较**：比较两种展开中 x² 项的系数，导出最终结果
4. **扩展讨论**：介绍欧拉证明的历史意义和更一般的公式
